# Air Quality Index (AQI) Prediction 🌍

This project utilizes Machine Learning to predict the Air Quality Index (AQI) based on historical environmental data. It aims to provide insights into pollution trends and help in forecasting air quality levels for better health and environmental management.

## 🚀 Features
* **Data Cleaning & Preprocessing:** Handling missing values and feature scaling for environmental datasets.
* **Exploratory Data Analysis (EDA):** Visualization of pollution trends and correlations between variables.
* **Model Training:** Implementation of Regression models (e.g., Random Forest, XGBoost) to predict AQI.
* **Evaluation:** Assessment using metrics like RMSE, MAE, and $R^2$ score.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
* **Environment:** Jupyter Notebook / Python Scripts

## 📂 Project Structure
* `BDA analysis.py`: Script for exploratory data analysis and feature engineering.
* `train_aqi_model.py`: Main script for training and validating the ML model.
* `.gitignore`: Excludes large datasets and local environments.

## 📊 Results
The model is designed to analyze various pollutants like $PM_{2.5}$, $PM_{10}$, and $NO_2$ to predict the overall AQI with high precision.
